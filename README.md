 # Age Calculator App ⏳🎉

A clean and **well-structured** Age Calculator web app built with **HTML, CSS & JavaScript**.  
It calculates your age in **years, months, and days** based on your birth date. 🧾✨

## 🚀 Features

- ✅ Clean and readable code (Clean Code practices)  
- ✅ Input validation with error messages  
- ✅ Accurate age calculation using JavaScript `Date`  
- ✅ Responsive design for mobile & desktop  
- ✅ User-friendly interface with interactive elements  

## 🛠 Technologies

- HTML5  
- CSS3 (Flexbox & Responsive Design)  
- JavaScript (DOM & Date manipulation)  

## ⚡ How to Use

1. Open `index.html` in your browser  
2. Enter **Day, Month, Year** of birth  
3. Click **Calculate**  
4. See your age in **years, months & days**  
5. Errors are shown for invalid/missing inputs ❌
